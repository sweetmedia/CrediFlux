# Bienvenido a CrediFlux

CrediFlux es un sistema integral de gestión de préstamos y cobranzas diseñado para instituciones financieras y empresas de microcrédito.

## Características Principales

- **Gestión de Préstamos**: Creación, seguimiento y administración completa del ciclo de vida de préstamos
- **Cobranzas Automatizadas**: Sistema de recordatorios y seguimiento de pagos vencidos
- **Multi-Tenant**: Arquitectura que permite múltiples organizaciones en una sola instalación
- **Integración WhatsApp**: Comunicación directa con clientes vía WhatsApp Business API
- **Reportes y Analytics**: Dashboard con métricas clave y reportes personalizables
