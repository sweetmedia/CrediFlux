## Typography
- **Wordmark:** Custom serif typeface with green gradient (#738566 → #163300)
- **Spelling:** "Crediflux" (capital C, lowercase remainder)

## Icon Description
- Green gradient arc forming a "C" shape (~270°)
- Radiating sunburst rays (yellow → orange gradient) fanning outward from the opening
- Symbolizes: credit flow, financial growth, trust, and opportunity
